Все изменения во фронтенде делать через задачи для Агента 2, не вносить напрямую.

**Why:** Пользователь явно указал — фронтенд делает 2-й агент.

**How to apply:** При любых правках в `frontend/` писать задачу в файл для Агента 2, не редактировать файлы самостоятельно.
